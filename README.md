Automated Recon & Vulnerability Analysis Tool

A Python-based security tool that automates network reconnaissance, service enumeration, vulnerability intelligence, and attack path suggestions using Nmap.

Overview

This project simulates a real-world external security assessment workflow. It performs:

Network scanning
Service and version detection
Risk analysis
CVE mapping (basic)
Attack path suggestions
Report generation (Markdown)

The tool is designed to demonstrate both offensive security thinking and automation skills, making it ideal for entry-level cybersecurity roles.

Features
Automated Nmap scanning
Service and version detection
Risk scoring engine
Structured Markdown report generation
Attack suggestions based on detected services
Basic CVE lookup from service versions
Clean CLI output (SOC-style)
